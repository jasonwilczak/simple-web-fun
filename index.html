How it Works

There are several key elements to the structure:

  • The HTML code for the anchor tag itself: <a href="">This is what shows on the page</a>
  • Everything that is needed goes inside the "href" attribute
  • The format looks like: mailto:to-field?subject=email-subject&cc=cc-field&bcc=bcc-field&body=body-text
  • to,cc, and bcc fields can have multiple email address, comma separated
  • subject and body fields should be url encoded, you can write out what you want and encode it using this tool: URL Encoder
